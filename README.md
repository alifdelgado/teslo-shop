# Teslo API

1. Clone project
2. Install packages

```bash
# npm i
or
# yarn install
```

3. Add your environment variables
4. Set up the database

```bash
docker compose up -d
```

5. Execute the seed

```bash
http://localhost:3000/api/seed
```

6. Start the project:

```bash
# npm run start:dev
or
# yarn start:dev
```
